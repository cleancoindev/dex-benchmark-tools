Glossary

Feasibility
A problem instance is feasible when it has a non-trivial solution. Otherwise it is infeasible.
Proving feasibility
A solver proves feasibility for a problem instance if it is able to find at least one nontrivial solution for it.
Proving infeasiblity
A solver proves infeasibility for a problem instance if it is able to prove that no nontrivial solutions exist for it.
Deciding feasibility
A solver decides feasibility for a problem instance if it is able to prove either feasibility or infeasibility.
Optimality
A solution is optimal if there is no other solution with higher utility.
